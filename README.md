# Vehemence Squadron

Prototype de rail shooter space opera en Three.js + Vite.

## Lancer le projet

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Les notes de conception et l'état du prototype sont dans [CONTEXTE.md](CONTEXTE.md).
